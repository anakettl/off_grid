# Projeto usando uma placa fotovoltaica de 20W

# Indicação
* Carregar aparelhos e lâmpadas de baixo consumo até 12v.
Não foi utilizado inversor de corrente nesse projeto pela baixa capacidade de geração de energia. 

# Capacidade
Painel: 8.33A
Bateria: 3.5A
